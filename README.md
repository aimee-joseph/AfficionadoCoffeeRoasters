# Product Optimization & Revenue Contribution Analysis
## Afficionado Coffee Roasters

## Overview:

Built an end-to-end product performance and revenue contribution analysis dashboard for Afficionado Coffee Roasters. This project identifies top-performing and underperforming products, quantifies revenue contribution by product and category, and measures revenue concentration across 
the menu to support data-driven menu optimization.

## Deliverables:

- Power BI Dashboard
- Streamlit Web App

## Key Features:
- Dynamic sidebar filters - category, product type, store location
- Top N product slider - adjustable from 1 to 20
- Pareto (80/20) revenue concentration analysis
- Hero vs long-tail product segmentation
- CSV export for product drill-down table
